## How to Use This Book

1. **Part I — Rust Ecosystem:** Start here. Essential terminology and tools with Node.js equivalents. Build your mental model before diving in.
2. **Part II — Developer Roadmap:** The main content. Follow the phased learning path from environment setup to backend development.
3. **Appendix:** Quick reference table and extended term definitions. Use for lookup when you need a reminder.

---

## Structure

| Part | Purpose |
|------|---------|
| **I. Rust Ecosystem** | Core tools, packages, build concepts, language fundamentals |
| **II. Developer Roadmap** | 16 phases, day-by-day learning from Week 1 to production |
| **Appendix** | Quick reference cheat sheet + extended terms (Lifetimes, Generics, Smart Pointers, etc.) |
