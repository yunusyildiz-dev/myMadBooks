# A. Quick Reference Cheat Sheet

| Rust Term   | Meaning                    | Node.js Equivalent     |
|-------------|----------------------------|-------------------------|
| rustup      | Toolchain manager          | nvm                     |
| rustc       | Compiler                   | tsc                     |
| cargo       | Build + package manager    | npm                     |
| rustfmt     | Code formatter             | prettier                |
| clippy      | Linter                     | eslint                  |
| crate       | Package / module unit      | npm package             |
| crates.io   | Package registry           | npmjs.com               |
| Cargo.toml  | Project manifest           | package.json            |
| Cargo.lock  | Locked dependencies        | package-lock.json       |
| Workspace   | Multi-package repo         | npm workspaces          |
| Trait       | Behavior interface         | TypeScript interface    |
| Ownership   | Memory model               | GC (conceptual diff)    |
| Macro (`!`) | Compile-time code gen      | Babel plugins           |

> **Tip:** Run `rustc --explain E0XXX` for detailed compiler error explanations.
