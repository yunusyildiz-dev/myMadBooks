## How to Use This Book

1. **Introduction:** Start here. The quest to give machines a mind — why operating systems matter and what this journey covers.
2. **Parts 1–8:** The main narrative. Follow the chronological path from ENIAC to cloud computing, Unix to Linux, CP/M to iOS and Android.
3. **Conclusion:** A reflection on the story still being written.
4. **Appendix:** Further reading and key links for deeper exploration.

---

## Structure

| Part | Era | Content |
|------|-----|---------|
| **I** | 1940s–1950s | The First Light — ENIAC, Von Neumann, batch processing |
| **II** | 1960s | The Birth of Multiplicity — Time-sharing, CTSS, Multics, Unix |
| **III** | 1970s–1980s | Personal Computer Revolution — CP/M, DOS, Mac, Windows |
| **IV** | — | Free Software & Linux — GNU, Linus Torvalds |
| **V** | 1990s | The Internet Age — Windows 95, Apple's dark period |
| **VI** | 2000s–2010s | Mobile Revolution — Symbian, iPhone, Android |
| **VII** | 2010s onward | Cloud, Containers & Future — AWS, Docker, Chrome OS, AI |
| **VIII** | — | Turkey and Operating Systems — Pardus |
